# 📸 Pexels App


## 🚀 Key Features

*   **🎨 Modern UI** — Fully built with Jetpack Compose, featuring Edge-to-Edge support and a native Dark Mode.
*   **🔍 Smart Search** — Keyword-based search functionality with dynamic featured collections.
*   **💾 Advanced Caching** — Robust offline support using Room database integrated with Paging 3.
*   **⚡ Pro Performance** — Smooth Staggered Grid, image zooming, and elegant loading animations.

---

## 🛠 Tech Stack

The application demonstrates the best practices of modern Android development:

- **Language:** [Kotlin](https://kotlinlang.org)
- **UI:** [Jetpack Compose](https://android.com)
- **Dependency Injection:** [Hilt](https://android.com)
- **Networking:** [Retrofit](https://github.io) & [Kotlinx Serialization](https://kotlinlang.orgdocs/serialization.html)
- **Local Database:** [Room](https://android.com)
- **Pagination:** [Paging 3](https://android.com)
- **Image Loading:** [Coil](https://github.io)
- **Architecture:** MVVM, Clean Architecture, Use Cases, Repository Pattern

## ⚙️ Installation

1. Clone the repository
2. Open the project in **Android Studio**.
3. Add your [Pexels API Key](https://pexels.com) to the `NetworkModule`.
4. Build and Run!
